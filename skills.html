Mes compétences techniques:


L'école WebForce m'a bien appris à structurer mon code HTML, ne pas hésiter à le commenter, et surtout, de respecter les normes en vigeurs.

Le CSS et la possibilité de donner son propre style à une page html. La possibilité de pouvoir faire des animations m'intéresse particulièrement !

Le JavaScript est une partie que j'ai beaucoup apprécié. La possibilité de pouvoir créer des interactions et des évènements dynamiques.

jQuery, la bibliothèque JavaScript libre et multiplateforme créée pour faciliter l'écriture de scripts côté client dans le code HTML des pages web.

PHP et la création/communication avec des bases de données. C'est là que tout prend son sens à ce moment.

Bootstrap est une collection d'outils utiles à la création du design de sites et d'applications web. C'est un ensemble qui contient des codes HTML et CSS, des formulaires, boutons, outils de navigation et autres éléments interactifs, ainsi que des extensions JavaScript en option.

Symfony est le framework que j'ai le plus utilisé pendant la formation. Je continue aujourd'hui à m'y interêssé, et je suit régulièrement les nouveautés sur le site officiel qui est très bien fait.

Le CMS ou système de gestion de contenu WordPress. Il est très intuitif mais a ses limites lorsque l'on veut traîter beaucoup de données. Plus on y associe de plugin, plus l'affichage du site sera lent. Très opérationnel pour créer des blogs.


Logiciels de codages:


Je code principalement avec Visual Studio Code qui est un éditeur de code extensible développé par Microsoft pour Windows, Linux et macOS. Les fonctionnalités incluent la prise en charge du débogage, la mise en évidence de la syntaxe, la complétion intelligente du code, les snippets, la refactorisation du code et Git intégré.

Malheureusement payant, PhpStorm est un éditeur pour PHP, HTML, CSS et JavaScript, édité par JetBrains. Il est très intérêssant de l'associer avec Symfony.
